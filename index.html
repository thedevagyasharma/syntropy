01 — The System

The rules came first.
The patterns followed.

Each layer operates on simple binary rules. The squares layer makes 9 decisions: each cell black or white. The rotated squares layer makes 4 decisions: each diamond present or absent.

Alone, each layer is unremarkable. When combined, they produce 8,192 distinct patterns, each one different, some of them beautiful.
